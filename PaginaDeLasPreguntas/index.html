Es un lenguaje de hipertexto que se usa para poder crear la estructura de las paginas web.

Las etiqutas son codigos los cuales son los que indican como va a ser la estructura de la pagina web para poder mostrar el contenido, y pues para poder usarlas se ocupa los signos de mayor y menos que "<>" y se pone la etiqueta que se desea usar sea como ejemplo la etiqueta h1 o p.

  • h1 a h6: Son encabezados de diferentes niveles.
  • p: Parrafos
  • ul: lista desordenada
  • ol: Lista ordenada
  • li: Elemento de lista
  • div: contenedor generico para agrupar elementos

CSS es un lenguaje de hojas de estilo que se utliza para poder darle apariencia a una pagina web.

son las cuales definen el tamaño, la posicion, la visibilidad y el estilo de los elementos de un documento.

El modelo de caja CSS es un módulo CSS que define cajas rectangulares, incluyendo sus rellenos y márgenes, que son generadas para los elementos y que se disponen de acuerdo al modelo de formato visual.

El diseño responsivo es una técnica de diseño web que se adapta a diferentes tamaños de pantalla. Se logra con CSS y HTML, y es la forma predeterminada de construir sitios web. y Para lograr adaptarse a las exigencias de cada tamaño de pantalla se utilizan “media queries” y hojas de estilo que indican las CSS (Cascading Style Sheets – Hojas de estilo en cascada) necesarias a tomar en función de cada tamaño de pantalla.

JavaScript es un lenguaje de programacion el cual se usa para poder hacer paginas web mas intactivas y dinamicas. y usualmente se integra con HTML y CSS

El DOM es como un puente que conecta el HTML, el CSS y el JavaScript. Permite que JavaScript acceda y modifique la estructura y el estilo de una pagina web y que respona alas instrucciones dadas por el usuario.

  • Interactuar con Formularios: sea como validar campos de un formulario.
  • Crear animaciones y efectos visuales: sea un menu desplegable.
  • mostrar y ocultar elementos: como una ventaja emergente la cual aparece al darle click a un boton.

Git es un sistema de control de versiones que se utiliza para gestionar el codigo. y es importante ya que permite hacer un seguimiento del codigo y conbinar cambios y volver a versiones anteriores, y permite a varias personas trabajar de manera conjunta y de manera no lineal.

Git es una herramienta fundamental para el desarrollo web moderno. Permite gestionar el código de manera eficiente, facilita la colaboración y garantiza la integridad del proyecto.

GitHub es una plataforma de desarrollo colaborativo que aloja repositorios de código utilizando el sistema de control de versiones Git. En esencia, es un lugar en la nube donde los desarrolladores pueden almacenar, gestionar y colaborar en sus proyectos de software.
